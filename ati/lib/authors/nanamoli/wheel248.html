# 《佛陀关于业的教导：来自《中部》的四部经典》分段整理

## 前言：业与日常生活

### 业无处不在

  • 业（kamma）关乎每个人：我们每天都在造业，清醒时更是如此。
  • 日常抉择的业：
  • **起床**：精进起床是善业，懈怠或勉强起床是恶业
  • **饮食**：贪著食物是恶业（贪根）
  • **帮助他人**：出于同情帮助病者是善业
  • **交通**：因公交车迟到而烦躁是恶业
  • **工作**：对他人不耐烦、愤怒或威胁，是越来越重的恶业；而慷慨、友善则是殊胜的善业
  • **回家**：踩到脚而咒骂是恶业，但立刻觉知“没有正念”则是善业
  • **晚间**：照顾病者、与孩子玩耍、讲本生故事（Jataka stories）是善业；疲惫时打开收音机/电视却心不在焉、暴食导致昏沉是恶业；而礼敬佛像、诵经、禅修则全是善业

### 业的分类标准：三不善根与三善根

#### 三不善根（恶业之根）

  • **贪根（greed-rooted）**：如暴食
  • **嗔根（hatred-rooted）**：如被踩脚后咒骂
  • **痴根（delusion-rooted）**：如工作中昏沉、白日梦、不能如实观照

**核心教导**：当心中生起任何烦恼状态，当这三种“不善根”掌控时，必定造作恶业。恶业一旦造下，无法消除或改变，终将在某时结果。恶业的果报从不带来快乐，而是痛苦、焦虑、沮丧或机会的局限。

#### 三善根（善业之根）

  • **无贪（non-greed）**：慷慨、出离
  • **无嗔（non-hate）**：慈爱、悲悯
  • **无痴（non-delusion）**：智慧、了知

**核心教导**：追求快乐须造善业。不论目标是现世幸福、来世善趣，还是通过证悟涅槃以终结生死轮回，造善业都不会错。

**关于不信者**：不信业与业果的人，无论信与否，依然在造业，也必然承受果报。关键在于“行”，而非“信”。

> **“行善得善，行恶得恶。”**

## 第一部经：《中部·第57经·狗行者经》（Kukkuravatika Sutta）

### 经文背景

佛陀时代有一些奇特修行者，例如模仿狗和牛的行为以求解脱。本经中，佛陀遇到两位这样的行者：

  • **布那（Punna）**：拘利耶人，行“牛戒”（ox-duty ascetic）
  • **斯尼亚（Seniya）**：裸体外道，行“狗戒”（naked dog-duty ascetic）

地点：拘利耶国的哈利达瓦萨那城（Haliddavasana）。

### 核心对话

#### 第一轮问答

布那问佛陀：斯尼亚长期行持狗戒，他的来世去向如何？

佛陀三次回避回答，但布那一再追问，佛陀终于作答：

**若有人圆满修持狗戒、狗习、狗心、狗行为——**

  • 身坏命终后，投生到狗群中
  • 但若他持邪见：“凭此戒、此苦行、此梵行，我将成为天神”——则必落两种去处：地狱或畜生道

**结论**：若狗戒修得圆满，导向狗群；若未圆满，导向地狱。

#### 斯尼亚的反应

斯尼亚听后流泪，声称不是因为佛陀的言论而哭，而是因为自己长久修持狗戒。他转而问布那的果报，佛陀给出类似回答：修牛戒圆满则投生牛群，否则堕落地狱。

#### 两人的皈依

  • 布那闻法后，当即皈依佛、法、僧，成为终身在家居士
  • 斯尼亚则请求出家受具足戒

佛陀告知：其他外道欲在此法、律中出家，需经四个月别住观察期。斯尼亚自愿观察四年。最终他出家受戒，精进独处，不久证得阿罗汉果。

### 四种业的分类（核心教义）

佛陀以自证智宣说四种业（kamma）：

1. **暗业有暗报（dark kamma with dark ripening）**

  • 定义：造作与恼害相应的身业、语业、意业
  • 果报：投生到有恼害的世界，受恼害的触（contacts），感受纯苦
  • 例证：地狱众生
  • 要义：“众生是业的继承者”

2. **明业有明报（bright kamma with bright ripening）**

  • 定义：造作不与恼害相应的身业、语业、意业
  • 果报：投生到无恼害的世界，受无恼害的触，感受纯乐
  • 例证：遍净天（Subhakinha）众生

3. **暗明业有暗明报（dark-and-bright kamma with dark-and-bright ripening）**

  • 定义：造作既与恼害相应又不与恼害相应的身、语、意业
  • 果报：投生到既有恼害又无恼害的世界，感受苦乐参半
  • 例证：人类、部分天神、部分恶道众生

4. **非暗非明业、有非暗非明报、导向业尽（neither-dark-nor-bright kamma）**

  • 定义：对于前三类业，生起**舍离的意志（volition）**——即舍离暗业、明业、暗明业的意志
  • 果报：超越业的范畴，导向业的灭尽

## 第二部经：《中部·第135经·小业分别经》（Cula-kammavibhanga Sutta）

### 经文背景

地点：舍卫城祇树给孤独园

问法者：**须婆（Subha）**，婆罗门，得耶（Todeyya）之子

### 须婆的问题

“人类之间为何有高低贵贱之分？为何有人短命、有人长寿，有人多病、有人健康，有人丑陋、有人美丽，有人无权、有人有权，有人贫穷、有人富裕，有人出身低贱、有人出身高贵，有人愚痴、有人智慧？”

### 佛陀的回答：业是根本原因

> **“众生是业的所有者，业的继承者，业为起源，业为亲属，业为皈依处。业使众生有高低贵贱之分。”**

### 十四对因果对应关系（列表）

| 行为（因） | 果报（此世若生为人） |

|-----------|-------------------|

| **杀生** | 短命 |

| **不杀生（慈悲一切众生）** | 长寿 |

| **用手、土块、棍棒、刀伤害众生** | 多病 |

| **不伤害众生** | 健康 |

| **易怒、心怀怨恨、表现嗔恚** | 丑陋 |

| **不易怒、不怀怨、不现嗔恚** | 美丽 |

| **嫉妒他人所得、恭敬、供养** | 无权/卑微 |

| **不嫉妒** | 有权/有影响力 |

| **不布施（食物、衣物等）给僧侣与婆罗门** | 贫穷 |

| **布施** | 富裕 |

| **傲慢、不礼敬应礼敬者** | 出身低贱 |

| **谦逊、礼敬应礼敬者** | 出身高贵 |

| **拜访僧侣/婆罗门时不问善与不善** | 愚痴 |

| **拜访时请问：“什么是善？什么是不善？什么是应做的？”** | 智慧 |

**重要说明**：经文同时指出，这些行为若造成恶果，可能先堕恶道，再回人道时受相应果报。

## 第三部经：《中部·第136经·大业分别经》（Mahakammavibhanga Sutta）

### 经文背景

地点：王舍城竹林精舍

起因：比丘**三弥提（Samiddhi）**与游方者**波塔利普塔（Potaliputta）**的对话。

波塔利普塔声称从佛陀处听闻：“身业是虚妄的，语业是虚妄的，只有意业是真实的。”三弥提否认此说，并指出：“造作有意的身、语、意业，其果报是苦。”

### 佛陀对三弥提的批评

佛陀指出三弥提的回答不准确（未经分析即作答），正确回答应是：

  • 造作身、语、意业→若其果报是乐，则感受乐
  • 若其果报是苦，则感受苦
  • 若其果报是不苦不乐，则感受不苦不乐

### 四种人的现象及常见误解

佛陀宣说世间存在四种人：

| 类型 | 行为模式 | 此世观察到的投生结果 |

|------|---------|-------------------|

| ① | 杀生、偷盗、邪淫、妄语、恶口、两舌、绮语、贪欲、嗔恚、邪见 | 堕恶道 |

| ② | 同上 | 生天善趣 |

| ③ | 不杀生...具正见 | 生天善趣 |

| ④ | 同上 | 堕恶道 |

### 对片面观察的批判

**佛陀指出**：有修行者以天眼见到①或④的情况后，可能错误地断言：“行恶者必然堕地狱”或“行善者必然生天”。这种绝对化的论断是错误的，因为它基于不完整的观察。

佛陀对四种片面观点的评价：

| 观察者观点 | 佛陀承认的部分 | 佛陀不承认的部分 |

|------------|--------------|----------------|

| “有恶业，有邪行之果报” | ✓ | — |

| “行恶者永远堕地狱” | — | ✗ |

| “有善业，有善行之果报” | ✓ | — |

| “行善者永远生天” | — | ✗ |

| “没有恶业” | — | ✗ |

| “没有善业” | — | ✗ |

### 佛陀的大业分别（核心教义）

导正上述误区后，佛陀给予完整的教导：

#### 四种人的业力分析

**① 行恶者堕恶道**：虽然此生造恶，但此前或此后可能造过善业，或在临终时生起正见。正是这些善业导致恶道苦报之前，他先投生善道。但此生的恶业仍会在某个时候成熟（现世、来生或以后）。

**② 行恶者生善道**：此生虽造恶，但此前或此后曾造善业，或在临终时生起正见。这些善业使他生在善道。但他此生的恶业仍将未来成熟。

**③ 行善者生善道**：同上逻辑，善业成熟导致善果。

**④ 行善者堕恶道**：此前或此后曾造恶业，或在临终时生起邪见。恶业先成熟。但此生的善业仍将未来成熟。

### 四种业的最终分类

佛陀总结为四种业：

1. **有业无能力（产生善果）且显现为无能力**（如①中暂时的恶果）

2. **有业无能力（产生善果）但显现为有能力**（如②中暂时的善果）

3. **有业有能力（产生善果）且显现为有能力**（如③中暂时的善果）

4. **有业有能力（产生善果）但显现为无能力**（如④中暂时的恶果）

第 2 部分

### **关于业的四种情况（注释14详解）**

根据注释14，佛陀在经中总结了一段极为精炼的开示，阿难尊者对此完全理解（或可能请求过解释）。注释进一步阐明了四种关于“业力”与“结果”的相互关系：

  • **① 强不善业（incapable of good result）**：一种强大的不善业，其果报会先于其他较弱的、未成熟的恶业成熟。这种业自身看起来“无能生善果”，但它的力量足以先产生苦报。
  • **② 善业（capable of good result）被不善近死业阻碍**：原本看似能产生善果的业，因为临终时出现的不善业（死亡近业，death-proximate kamma）而变得暂时“无能”，无法立即产生善报，恶报先成熟。
  • **③ 强善业（appears capable of good result）**：一种强大的善业，即便有大量累积的不善业存在，它仍然能先成熟，产生善果。
  • **④ 不善业（incapable of good result）被善近死业扭转**：原本看似只能产生恶果的不善业，因为临终时出现的善业（死亡近业）而变得能够产生善果——善近业先成熟。

**关键引用**：此段是佛陀在《相应部·受相应·独处品》中的开示（注2），且涉及四种回答问题的方式（注1：直接回答、反问、反问式回答、沉默）。注释中特别提及了提婆达多（Devadatta）和“铜牙”（Coppertooth）等故事作为例证（见注7、9），说明临终心念的巨大作用。

### **《萨罗村婆罗门经》（Majjhima Nikaya 41, Saleyyaka Sutta）**

#### **引言与背景**

  • **问题**：萨罗村的婆罗门在家众问佛陀：“是什么原因，什么条件，使得一些众生身坏命终后堕入恶趣（地狱、畜生、饿鬼），而另一些则投生到善趣（天界）？”
  • **佛陀的回答**：直接指出——原因在于**非法、非正行（不正当的身语意业）**导致恶趣；**法、正行（正当的身语意业）**导致善趣。

#### **一、十不善业道（导致恶趣）**

佛陀详细列举了三种身不善业、四种语不善业、三种意不善业：

  • **身三不善业**：
  • 杀生（残忍、血腥、无情）
  • 偷盗（取他人未与之物）
  • **邪淫**（与受母亲、父亲、父母、兄弟、姊妹、亲属、有夫之妇、受罚者、已受聘礼者等保护的女性发生关系）
  • **语四不善业**：
  • **妄语**（明知而说假话，为自己、他人或微小利益）
  • **离间语**（说使人和睦破裂的话，喜欢争斗）
  • **恶口**（粗暴、伤人、令人愤怒、无法入定的话语）
  • **绮语**（说不合时宜、不真实、无益、非法、非律、无记录价值、无理由、无范围、不导向善的话语）
  • **意三不善业**：
  • **贪欲**（贪求他人财物：“唉！他的东西是我的就好了！”）
  • **瞋恚**（心怀恶意：“愿这些众生被杀害、被消灭！”）
  • **邪见**（扭曲的看法：认为没有布施、供养、祭祀的果报；没有善业恶业的果报；没有此世他世；没有父母；没有化生众生；没有如实知见此世他世的善戒行者）。此邪见否定一切道德价值（注2：与唯物断见类似）。

#### **二、十善业道（导致善趣）**

对应地，三种身善业、四种语善业、三种意善业：

  • **身三善业**：
  • 离杀生（放下刀杖，仁慈、有同情心）
  • 离偷盗（不取他人未与之物）
  • 离邪淫（不与非伴侣者发生关系，包括受保护的女性和有夫之妇等）
  • **语四善业**：
  • 离妄语（不说假话，如被传唤作证时如实回答）
  • 离离间语（不作离间，而是促进和睦、喜爱和谐）
  • 离恶口（说柔和、悦耳、入心、有礼、众人喜爱的话）
  • 离绮语（说合时、真实、有益、合法、合律、有记录价值、有理、有范围、导向善的话）
  • **意三善业**：
  • **无贪**（不贪求他人财物）
  • **无瞋**（心怀善意：“愿众生无怨、无恼、无苦，快乐生活！”）
  • **正见**（相信布施、供养、祭祀有果报；善业恶业有果报；有此世他世；有父母；有化生众生；有如实知见此世他世的善戒行者）

#### **三、持正行者的临终愿望与投生可能**

佛陀进一步说明：一位持正行（遵守十善业道）的在家居士，可以在临命终时发愿投生到不同层次，并因其正行之力而有可能实现。发愿的对象从低到高依次包括：

  • **人间富贵**：投生为有大财富的刹帝利、婆罗门、居士（第15-17段）
  • **六欲天**（从低到高）：
  • 四大天王天
  • 三十三天
  • 夜摩天
  • 兜率天（知足天）
  • 化乐天
  • 他化自在天（第18-23段）
  • **色界天**（初禅至四禅）：
  • 梵众天
  • 光天（少光天、无量光天、光音天）
  • 净天（少净天、无量净天、遍净天）
  • 广果天（非常丰硕果报天）
  • **净居天**（五不还天）：自住天、无烦天、善见天、善现天、色究竟天（第24-38段，注意此处名称有差异，但对应五净居天）
  • **无色界天**：
  • 空无边处天
  • 识无边处天
  • 无所有处天
  • 非想非非想处天（第39-42段）
  • **究竟涅槃**：发愿现世证得无漏心解脱、慧解脱，成为阿罗汉（第43段）

**关键说明**：发愿本身并非万能，必须建立在“持正行”的基础上。例如，一个吝啬者发愿富贵，其业力却会带来贫穷。若想投生梵天，还需修习禅定（jhana）；若想投生净居天，需达到不还果（三果）；若想证涅槃，则需修行与智慧。

#### **四、经末与皈依**

萨罗村的婆罗门听了佛陀的开示后，大受感动，赞叹佛陀“把颠倒的扶正，揭示隐藏的，为迷途者指路，在黑暗中举灯”。他们随后**皈依佛、法、僧**，成为终身优婆塞（在家弟子）。

**注释补充**：

  • 注1：关于“化生众生”（spontaneously born beings）——指因业力而突然出现的生命形态，如天神、饿鬼、地狱众生（参见MN 12《师子吼大经》）。
  • 注2：关于邪见——佛陀时代某些外道所持的否定道德因果的观点，如“无因无缘论”、“无作用论”等（参见雷迪尊者《八正道及其因素》）。

**总结**：本经系统阐述了业力法则的核心——十善业与十不善业的具体内容，并说明持正行不仅可以避免恶趣，还能通过临终发愿实现从人间到天界乃至涅槃的各种善果。这是一部兼具理论定义与实践指引的经典，强调道德行为（戒）是解脱的基础。
