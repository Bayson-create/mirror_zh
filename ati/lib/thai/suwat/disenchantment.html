祛魅

Disenchantment

全文现代汉语翻译
本页为短篇全文翻译，保留原文段落、列表、引文和链接。

我们现在开始禅修，就像我们每天所做的那样。我们必须将此视为一个重要机遇。尽管我们的修行还没有达到令我们满意的法，但至少这是一个开始，一个重要的开始，可以聚集心的力量，使我们的念住、定力和明辨变得健康和成熟。我们应该尝试将这些品质聚集在一起，以便它们能够相互加强，洗净我们心中的污点、烦恼——因为当烦恼生起时，它们不会给心灵带来平静、纯洁或休息。恰恰相反：它们会导致痛苦、不安和骚乱。它们阻挡了任何了解或看见佛法的内观。没有任何烦恼可以鼓励我们修习佛法、了解或见到佛法。它们只会妨碍我们的练习。

因此，无论什么心理状态妨碍了我们的修行，我们都应该将其视为烦恼——因为烦恼不会自行漂浮。他们必须依赖头脑。任何困倦或懒惰的心理状态，任何焦躁、愤怒或烦躁的心理状态：这些都是烦恼。它们是受烦恼影响、被烦恼克服的心理状态。

如果我们内心出现任何这些心理状态，我们应该觉知它们。当心困倦的时候，我们应该让它保持清醒 布多 如此，它就会醒来，摆脱困倦。当心不安、烦躁时，我们应该用明辨去省察事情，看看这些心境是没有任何作用的。然后我们应该迅速回到我们的定力练习上，将心牢固地植根于我们的禅修主题中，不要让心再次焦躁不安和散乱。

我们将心集中在觉知它的禅修词上， 布多 ——什么是意识到的，什么是清醒的。我们牢记它，就像它是一根牢牢地扎在地上的柱子一样。不要让思绪偏离你所关注的基础岗位。但无论你的焦点是什么，都不要让你的焦点紧张。当你注意力集中时，你必须保持良好的心情。以念住和明辨的态度来做这件事，而不是妄想的态度，想要知道这个或看到那个，或者强迫事物与你的想法一致。如果你这样打坐，你的心情就会变得紧张，你就无法打坐太久。很快你就会开始变得烦躁。

所以如果你想长时间打坐，你必须保持中立，以舍为基础。如果你想要知识，就坚定地专注于你已经知道的事情。保持你的思想坚定。找到一种方法可以帮助您保持专注而不会走神。例如，努力保持你的思想坚定专注，并运用你的观察和评估能力来建立你的想法的基础。 佛陀。 所有这些事情都必须聚集在同一个地方，连同你需要做的任何想法一起，这样念住就不会消失，让不熟练的外部问题闯入，或者为内心的关注留下一个开口，或者让自己被过去的想法所困扰——你今天早些时候，或者很多天，很多月，很多年前知道的、看到的、说过的或做过的事情。你必须专注于当下。

如果你已经采取了 布多 作为你的禅修主题，不断地一遍又一遍地回到它。 布多 代表意识。如果你能保持意识不流失，这将会产生重要的影响。如果你把呼吸作为你的主题，你就必须在每次呼吸的进出时保持觉知。你不能让自己迷失方向。除了呼吸之外，你什么都不能作为念住的焦点。无论哪种情况，都适用相同的原则。你们做同样的事情，唯一的区别是你们意识的主题。

为什么佛陀教导我们专注于呼吸？因为我们不必寻找它，不必猜测它，不必思考它的存在。这是目前的现象。没有过去的呼吸或未来的呼吸这样的事情。只是当下的呼吸进进出出。这就是为什么它适合锻炼我们的念住，适合将我们的念住和觉知聚集在一处，适合牢固地建立定力。

因此，你可以专注于任一主题——无论你已经禅修过哪个主题，并发现念住可以快速建立而不会消失，并且可以快速产生一种宁静与平和的感觉。将该主题设置为您的基础。当您开始时，请重点记住该主题。

Once the mind has had enough stillness, if you simply want it to become more still, the mind will get into a state where it isn't doing any work because it's not distracted in any way.如果出现这种情况，你就必须开始思考了。在念住的基础上，我们被教导要思考身体的各个方面。我们不必考虑任何其他事情。如果你想从无常的角度去思考，就在这个身体里。如果你想从压力的角度去思考，压力就在这个身体里。你可以从任何角度思考它。如果你想从断除贪爱的角度去思考，你可以看那些肮脏、恶心的东西，你会发现它们充满了身体。这是需要你运用自己的智慧的事情。无论你用什么角度，你都必须去观察事物，让它们变得更加微妙和精致。一遍又一遍地思考它们，直到你清楚地看到事物，从而产生 尼比达, 或祛魅，这样你就不会像以前那样迷惑地执着于事物并赋予它们意义。

转变新的思维，将你的观点变成新的观点。你不再想要你以前的错误观点。转变你的旧见，生起正见。转变你旧有的思维方式，产生正确的决心——将身体视为令人厌恶和没有吸引力的。这是 涅克哈玛-桑卡帕， 出离的决心，摆脱欲欲的决心。我们不会朝其他方向思考，也不会朝其他方向漫游。我们试图逃避身体是美丽的观点。眼睛看到的身体只是外层皮肤。从来没见过里面的脏东西。尽管它可能时不时地看到它们，例如当有人在事故中死亡或当病人接受手术时，心中有一些东西使我们无法将其放在心上并生起辨别力。有些东西使我们无法以比肉眼所见更微妙的水平来思考事物。我们看到这些东西，然后就忽略它们。我们还没有达到足够深刻的水平，不足以引起幻灭。

所以观照身体。如果心已经发展了足够强大的基础，它就不应该仅仅停留在寂静的层面上。但如果你还没有达到那种静止的程度，你就不能跳过它。你首先必须让心静止，因为静止的坚实基础是绝对必要的。如果你在心平静下来之前尝试沉思，你就会产生只在你禅修期间才会持续的知识。当你离开禅修并且心不再坚定时，你的新理解就会消失。你旧的理解会回来，就像你从未禅修过一样。无论你过去以何种方式被欺骗，你都会再次被欺骗。无论你以前有什么看法，都不会改变。无论你以何种方式思考，只要你的新思维方式不是建立在寂静的基础上，你最终都会像以前一样陷入迷惑。

这就是为什么静止如此重要。我们必须让心从静止中获得力量，然后让它从身体的 32 个部分来观照身体本身。你可以选择任何一个部分，重点关注它，直到弄清楚为止。或者您可以专注于五个一组的部分。当你到达液体部分时，你可以以六个为一组集中注意力，因为总共有 12 个。你可以来回思惟它们——如果你的念住还没有修练到坚固的地步，那么来回思惟这些事情，就像导师教导新出家一样： 凯萨、洛马、纳卡、丹塔、炸玉米饼 （头发、体毛、指甲、牙齿、皮肤），然后将它们翻转过来 塔科、丹塔、纳卡、洛马、凯萨。 然后你可以继续下一组五个 - mansam、nharu、atthi、atthimiñjam、vakkam （肌肉、肌腱、骨骼、骨髓、脾脏）。这称为五组随观。

这就是我们开始练习念住的方式。当你以这种方式修习念住时，如果出现这五个部分中任何一个的视觉形象，抓住它并观照它，使它变得更深、更精致。观照它，直到你能将身体分成不同的部分，看到每一部分都如此。让你了解身体的里里外外，认识到其他众生也是如此。如果您想看看什么是不干净的，您可以在这里找到。如果你想看看什么是无我，你会在这里找到它。在你的脑海中反复思考这些事情，并问自己它们是否是恒定的。 What kind of pleasure is there in these things?值得还是不值得？经常关注这些问题，经常观察它们，直到你熟练为止，心最终会愿意接受真理，改变旧有的错误看待事物的方式，而以佛陀所教导的法来看待它们。

当你的观点经常以这种方式改变时，心就会体验到一种新的宁静与平和。它将远离贪、嗔、痴之火的狂热；转而转为念住、定力和明辨。它的知识和观点将变得清晰。就不会再动摇了。它会变得勇敢，不再像以前那样害怕——因为它已经知道了真相： 除了五蕴之外，没有任何事物会受到痛苦；除了元素之外，没有任何东西会死亡。 心就牢固地扎根了。它可以带着一种舒适的自信来禅修，而不用担心疼痛、疾病或任何事情。你可以把事情从头到尾分开。到那时就算死了，你也心满意足了，因为虽然死还没有来，这些东西已经自行分离了。你已经思考过它们，并看到了它们的本来面目，每一个。

所以我要求大家在思考的时候要有坚定的原则。真诚地去做——不要只是走过场——因为所有这些事情都是真诚的。如果我们不禅修，烦恼就会栖息在我们的思想中，欺骗我们，使我们看不到事物的真实面目。如果我们仅仅依靠我们的眼睛，它们就会欺骗我们。眼睛只能看到事物的外部。它看到皮肤，而皮肤可以通过化妆来欺骗我们。它能看到头发，并且可以通过化妆来欺骗我们。它能看到身体的毛发——比如眉毛和胡须，这些东西可以通过打扮来欺骗我们。它能看到手指甲和脚趾甲，可以通过化妆来欺骗我们。它看到牙齿，可以通过治疗来欺骗我们，以便我们对牙齿做出各种假设。眼睛没有辨别力。它让我们被欺骗—— 但这不是欺骗的原因。 头脑是在欺骗自己。一旦它欺骗了自己，它就会对自己做出各种各样的假设，从而陷入自欺欺人的境地。当它让自己遭受这样的痛苦时，就没有办法了。这是真实的事实。 清楚地知道心是自欺欺人的。 当它没有庇护所时，它就可以一直欺骗自己。

因此，我们必须培养心可以持守和皈依的品质，这样烦恼就无法继续欺骗它。 Look so that you can see more deeply through things.尝试分析事物，看看什么是假的，什么是经过修饰和伪装的。一旦你看到任何东西，你就会发现什么是假的、什么是编造的。你就会知道：“真正的东西没有这个颜色、这个气味、这个形状。”你会发现事情总是在变化。这称为以法的品质作为你的皈依处，作为你看、听、闻、尝和接触事物时要坚持的东西。你将拥有了解并看到事物本来面目的品质——所以他们将无法欺骗你。你无法欺骗自己， 因为你会感到羞耻。 心对自己、对其旧有的方式不再抱有幻想——为什么它还要再欺骗自己呢？可以看出，它并没有从这种行为中获得任何好处。

相反，您将看到它如何从新视图中真正受益。它们使头脑平静。清除。带着幸福感释放自我。所有沉重的旧负担都消失了。它不再贪图获得很多东西，因为不再有放纵。它不利用任何东西来放纵自己。它所需要的只是维持生活的四种必需品——这就足够了。它不需要投资任何东西。它在禅修带来的宁静中找到幸福和幸福。它曾经迷恋周围的事物，在不知不觉中变得无知：当它专注于真正了解这些事物时，它的妄想就会消失。无知消失了。心从这些事物中获得与它们实际相一致的知识。它醒悟过来，不再像以前那样迷恋这些东西，不再像以前那样误解它们。

这就是它的问题的终结。
