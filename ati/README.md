# Access to Insight 现代汉语镜像

本目录是 Access to Insight 英文离线镜像的现代汉语阅读版本，目标是在保留原网站目录结构、页面格式、内部导航、重要引文和来源链接的基础上，为中文读者提供可离线访问的佛法资料。

## 收录范围

- 网站首页、帮助、术语表和多维索引
- 上座部佛教基础介绍
- 文库及作者、译者目录
- Thai Forest（泰国森林传统）开示
- Study Guides（佛法学习指南）
- 非经藏文献、修行路径和技术说明页面
- 原网站所需的 CSS、JavaScript、图片、图标及其他公共静态资源

短篇文章原则上全文翻译；篇幅很长的文章依照原章节顺序整理为完整中文导读，保留核心论证、修行方法、重要譬喻、关键引文和来源链接，并提供英文原文入口。

## 明确排除

以下两类内容不纳入本项目的现代汉语化范围：

1. **经文翻译**：`tipitaka/` 下的巴利三藏原文及英文经文译本不进行中文翻译。现有页面可保留相关索引或英文原文链接，但不以本项目重新翻译经藏。
2. **历史公告**：`news/` 下的历史通讯、站务公告和旧版更新记录不进行中文镜像。

这两项缺失属于项目范围约定，不应计为待完成翻译。

## 使用方式

从 `佛法修行` 目录启动静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000/mirror_zh/ati/
```

主要入口：

- `ati/index.html`：网站首页
- `ati/lib/index.html`：文库
- `ati/lib/authors/index.html`：作者与译者
- `ati/lib/thai/index.html`：泰国森林传统
- `ati/lib/study/index.html`：学习指南
- `ati/help.html#map`：网站结构

## 翻译与校订说明

中文页面经过结构、链接和浏览器渲染检查，但机器辅助翻译仍可能存在人名、巴利语术语或佛学概念表达不统一的问题。正式引用或用于学术研究时，请通过页面上的英文原文入口核对原文。
