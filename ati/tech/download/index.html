处理下载的文件
© 2014

整个网站上散布着一些文件链接，供您下载到计算机上并离线阅读。以下是如何使用这些文件。

处理 PDF 文件

PDF文档 采用 Adob​​e 专有的便携式文档格式。它们包含精心设计的格式、图形和自定义字体。它们通常看起来比普通网页更“精致”——尤其是在打印时。

如果文件下载后没有自动打开，请双击该文件。如果这不起作用，请获取免费的副本 Adobe Acrobat 阅读器 软件。

使用 ZIP 文件

Zip 文件包含经过压缩以减少下载时间的文件集合。如果下载后存档没有自动解压，请尝试双击它。如果这不起作用，您可能需要获得一个便宜的实用程序来解压缩它。以下是一些建议：

  • 麦金塔用户: 填充膨胀器 （免费）或 齐皮特 （共享软件）。
  • Windows 用户： 7Zip （自由的）。
  • Unix、Linux等用户： 您可能已经拥有解压缩 ZIP 文件所需的程序。它因系统而异，但“unzip”或“uzip”等命令已知可以工作。 （“gunzip”是一种不同的动物，并且 不是 工作。）如果您不确定系统上的命令是什么，请输入“男人-k zip”来追踪它。如果您的系统上确实没有它，请让您的系统管理员为您获取它。

解压缩存档后，其中包含的文件将出现在硬盘上的文件夹中。如果这些文件是 HTML 文件，只需使用 Web 浏览器程序打开它们即可。如果它们是其他类型的文本文件，请使用您喜欢的文本编辑器打开它们。
